# The Midterm Project

An informational, nonpartisan voter education website for the 2026 U.S. midterm elections.

## What Is This?

A static website that helps voters understand:

- **How Congress works** - Senate, House, and the basics of midterm elections
- **Who's on the ballot in 2026** - Senate seats up for re-election, competitive House races, and special elections
- **Why it matters** - Path to majority, key issues, and why your vote counts

## Features

- Educational explainers on the Senate, House, and midterms
- Candidate profiles with photos, party, and key issues
- Interactive "swipe" explorer to browse candidates
- Competitive race breakdowns with ratings
- Special election coverage (Ohio, Florida)

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [React](https://react.dev/) - Interactive components
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Vercel](https://vercel.com/) - Hosting

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

## Commands

| Command             | Action                                       |
| :------------------ | :------------------------------------------- |
| `npm install`       | Install dependencies                         |
| `npm run dev`       | Start local dev server at `localhost:4321`    |
| `npm run build`     | Build production site to `./dist/`           |
| `npm run preview`   | Preview production build locally             |
